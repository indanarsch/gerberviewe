# appstrappy

## tools

* [Gulp](http://gulpjs.com/)
* [Coffeescript](http://coffeescript.org/)

## stack and such

* [AmpersandJS](https://ampersandjs.com/)
[Hapi](http://hapijs.com/)
* [Jade](http://jade-lang.com/)
